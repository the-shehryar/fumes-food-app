## рҹҡҖ Getting Started

### Prerequisites

- Node.js `18+`
- Expo CLI
- Appwrite Cloud or self-hosted instance
- Android/iOS device or emulator

### Installation

```bash
# Clone the repo
git clone https://github.com/the-shehryar/fumes-food-app.git
cd fumes-food-app

# Install dependencies
npm install

# Start the development server
npx expo start
```

> вҡ пёҸ Some features (mock location detection) require a **dev build** вҖ” not Expo Go.
> ```bash
> npx expo run:android
> ```

### Environment Variables

Create a `.env` file in the root (see `.env` already in repo вҖ” **do not commit real keys**):

```env
EXPO_PUBLIC_APPWRITE_ENDPOINT=https://cloud.appwrite.io/v1
EXPO_PUBLIC_APPWRITE_PROJECT_ID=your_project_id
EXPO_PUBLIC_APPWRITE_DATABASE_ID=your_database_id
EXPO_PUBLIC_APPWRITE_BUCKET_ID=your_bucket_id
```

## рҹҸ—пёҸ Architecture Highlights

### рҹ”„ Local-First Search Strategy
```
App Load    вҶ’  Fetch menus from Appwrite  вҶ’  Save to AsyncStorage
Search      вҶ’  Read AsyncStorage          вҶ’  Filter locally (instant вҡЎ)
            вҶ’  Cache miss?                вҶ’  Fallback to Appwrite
```

### рҹ–јпёҸ Image Pipeline
```
Pick Image  вҶ’  Compress (1024px / 0.7 quality)  вҶ’  Upload to Appwrite Storage
```

### рҹ”җ Secure Image Loading
```
Request image  вҶ’  Generate JWT  вҶ’  Download to device cache  вҶ’  Render from local URI
```

### рҹ“Қ Location + Mock Guard
```
App opens  вҶ’  Request location permission
           вҶ’  Check for mock location (turbo-mock-location-detector)
           вҶ’  Mock detected? Block + Alert
           вҶ’  Real location? Reverse geocode вҶ’ get city name
```
